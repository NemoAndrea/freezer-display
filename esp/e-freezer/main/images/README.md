# Images (c-array)

These images were converted from .bmp files (in [the assets directory of this repo](../../../../assets/)) using [LVGL's online converter tool](https://lvgl.io/tools/imageconverter). Be sure to use the converter for LVGL 8.0 and the `CF_INDEXED_4BIT` format with `C array` output.

The files are still very inefficient in terms of disk usage. Since we have enough memory, it is not considered a priority to address this.
